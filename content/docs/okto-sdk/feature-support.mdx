Okto provides a seamless and secure way to integrate blockchain functionality into your applications. The Okto SDK enables developers to onboard users, manage wallets, interact with multiple chains, and facilitate smooth crypto transactions.

---

## Platform Support

Okto offers a range of integration options tailored to diverse development platforms, ensuring you have the right tools for your project:
- [**React**](/docs/react-sdk) ‚Äì A React library for integrating Okto into web applications.
- [**NextJS**](/docs/nextjs-sdk) ‚Äì A NextJS library for integrating Okto into web applications.
- [**React Native**](/docs/react-native-sdk) ‚Äì A React Native SDK for mobile applications.
- [**TypeScript**](/docs/typescript-sdk) ‚Äì A TypeScript SDK for building Web3 applications. 
- [**Unity**](/docs/unity-sdk) ‚Äì A Unity SDK for blockchain-based gaming apps.
- [**Wagmi Adapter**](/docs/wagmi-integration) ‚Äì A connector to use Okto with the Wagmi hooks ecosystem.
- APIs ‚Äì For custom backend integrations.

The Okto SDK supports multiple blockchain ecosystems including EVM chains, Solana, and Aptos, enabling seamless cross-chain interactions.
