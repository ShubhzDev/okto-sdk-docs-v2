## Okto SDK

<div className="rounded-xl p-8 border mb-8">

The **Okto SDK** allows developers to leverage Okto's chain abstraction technology in their own applications. Available for React, React Native, Flutter, Unity, and TypeScript, it offers a simple path to add Web3 capabilities to any application.

<Callout type="info">
**Okto Wallet and Okto SDK are independent offerings:** Okto Wallet is a standalone DeFi app, while Okto SDK is a developer tool for building Web3 applications.
</Callout>
