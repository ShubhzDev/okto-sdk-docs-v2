## POST Request

### Endpoint

```bash
POST https://sandbox-api.okto.tech/api/oc/v1/estimate
```

### Headers

| Name | Type | Description |
|------|------|-------------|
| Content-Type | string | Must be `application/json` |
| Authorization | string | Bearer token for authentication |

<CollapsibleCallout type="tip" title="Request Parameters">
- `fromChainTokenAddress`: The encoded function call data for the token transfer
- `fromChainCaip2Id`: Gas limit for the actual function call
- `toChainTokenAddress` and `maxPriorityFeePerGas`: Gas fee parameters
- `toChainCaip2Id`: Unique identifier for the transaction
- `sameChainFee`: Address of the paymaster contract
- `sameChainFeeCollector`: Data required for paymaster operations
- `crossChainFee`: Gas limit for post-operation checks
- `crossChainFeeCollector`: Gas limit for paymaster verification
- `fromChainTokenAmount`: Gas required for pre-verification steps
- `minToTokenAmount`: Address of the transaction sender
- `slippage`: Gas limit for transaction verification
</CollapsibleCallout>
