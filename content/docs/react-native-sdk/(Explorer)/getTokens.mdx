The `getTokens` function fetches the list of supported tokens from the backend.

### Example

<Accordions defaultValue="Usage">
<Accordion title="Usage">
<Tabs items={['TypeScript']}>
<Tab value="TypeScript">
```typescript
import { useOkto } from '@okto_web3/react-native-sdk';
import { getTokens } from '@okto_web3/react-native-sdk'; // [!code highlight]

function TokensList() {
    const oktoClient = useOkto();

    async function fetchTokens() {
        try {
            const tokens = await getTokens(oktoClient); // [!code highlight]
            console.log('Supported tokens:', tokens);
        } catch (error) {
            console.error('Error fetching tokens:', error);
        }
    }

    return (
        <Pressable onPress={fetchTokens}>
            <Text>Fetch Supported Tokens</Text>
        </Pressable>
    );
}
```

## Get Tokens

<sub><i>async</i></sub> `getTokens(oktoClient: OktoClient)` retrieves the list of supported tokens.

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `oktoClient` | `OktoClient` | Instance of OktoClient obtained from useOkto hook |

### Response

<Callout title="Success Response">

| Field Name | Type | Description |
|------------|------|-------------|
| `tokens` | [`Promise<Token[]>`](./technical-reference#models) | Array of supported tokens |
