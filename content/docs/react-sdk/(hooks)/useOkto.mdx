The `useOkto` hook is the primary way to access Okto functionality within React components. It provides access to the Okto client instance, allowing you to manage authentication, user operations, and session management.

### Hook Overview

| Hook | Description |
|------|-------------|
| [`useOkto`](#useokto-hook) | Access the Okto client instance and its methods |

<div className="method-box">

## useOkto Hook

`useOkto()` returns an instance of OktoClient.

### Requirements

This hook must be used within components wrapped by the `OktoProvider`.

### OktoClient Properties and Methods

<Callout title="OktoClient Interface">

| Property | Type | Description |
|----------|------|-------------|
| `user`            | `User \| undefined` | Current authenticated user information |
| `userSWA`               | `Hex \| undefined` | User's Smart Wallet Address |
| `vendorSWA`            | `Hex \| undefined` | Vendor's Smart Wallet Address |

| Method | Signature | Description |
|--------|-----------|-------------|
| `loginUsingOAuth` | `Promise<User>` | Authenticate using OAuth providers |
| `verifyLogin` | `Promise<boolean>` | Verify if user is logged in |
| `getAuthorizationToken` | `Promise<string>` | Get current authorization token |
| `paymasterData` | `Promise<Hex>` | Get paymaster data for transactions |
| `executeUserOp` | `Promise<string>` | Execute a user operation |
| `signUserOp` | `Promise<UserOp>` | Sign a user operation |

</Callout>

```tsx
import { useOkto } from "@okto_web3/react-sdk";

export default function Sample() {
    const oktoClient = useOkto();
    
    // Access user state
    const isLoggedIn = !!oktoClient.user;
    
    // Example authentication
    async function handleLogin(googleCredential) {
        try {
            await oktoClient.loginUsingOAuth({
                idToken: googleCredential,
                provider: "google"
            });
        } catch (error) {
            console.error("Login failed:", error);
        }
    }

    // Example user operation
    async function handleTransaction(userOp) {
        try {
            const signedOp = await oktoClient.signUserOp(userOp);
            const txHash = await oktoClient.executeUserOp(signedOp);
            console.log('Transaction hash:', txHash);
        } catch (error) {
            console.error('Transaction failed:', error);
        }
    }

    return (
        <div>
            {isLoggedIn ? (
                <div>
                    <p>Welcome, User SWA: {oktoClient.userSWA}</p>
                    <p>Vendor SWA: {oktoClient.vendorSWA}</p>
                </div>
            ) : (
                <button onClick={() => handleLogin()}>Login</button>
            )}
        </div>
    );
}
```
