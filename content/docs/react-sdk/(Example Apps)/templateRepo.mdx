## Getting Started

For detailed installation instructions, select your framework below:

<Tabs items={["React", "Next.js"]}>
  <Tab value="React" label="React">

  <div className="max-w-[450px] my-5 mx-auto">
      <img 
          src="/images/react-template.png" 
          alt="React Template App" 
          className="w-full rounded-lg shadow-sm"
      />
  </div>

  1. Clone the repository:
  ```bash
  git clone https://github.com/okto-hq/okto-sdkv2-react-template-app.git
  ```

  2. Change into the project directory and install dependencies:
  ```bash
  cd okto-sdkv2-react-template-app
  npm install
  ```

  3. Set up environment variables:
  ```bash
  cp .env.sample .env.local
  ```

  4. Add your API keys to the `.env.local` file:
  ```
  VITE_VENDOR_PRIVATE_KEY="YOUR_VENDOR_PRIVATE_KEY"
  VITE_VENDOR_SWA="YOUR_VENDOR_SWA"

  VITE_GOOGLE_CLIENT_SECRET="YOUR_GOOGLE_CLIENT_SECRET"
  VITE_GOOGLE_CLIENT_ID="YOUR_GOOGLE_CLIENT_ID"
  ```

  5. Start the development server:
  ```bash
  npm start
  ```

  1. Clone the repository:
  ```bash
  git clone https://github.com/okto-hq/okto-sdkv2-nextjs-template-app.git
  ```

  2. Change into the project directory and install dependencies:
  ```bash
  cd okto-sdkv2-nextjs-template-app
  npm install
  ```

  3. Set up environment variables:
  ```bash
  cp .env.sample .env.local
  ```

  4. Add your API keys to the `.env.local` file:
  ```
  NEXT_PUBLIC_VENDOR_PRIVATE_KEY="YOUR_VENDOR_PRIVATE_KEY"
  NEXT_PUBLIC_VENDOR_SWA="YOUR_VENDOR_SWA"
  AUTH_SECRET="YOUR_AUTH_SECRET"
  NEXT_PUBLIC_GOOGLE_CLIENT_SECRET="YOUR_GOOGLE_CLIENT_SECRET"
  NEXT_PUBLIC_GOOGLE_CLIENT_ID="YOUR_GOOGLE_CLIENT_ID"
  ```

  5. Start the development server:
  ```bash
  npm run dev
  ```
