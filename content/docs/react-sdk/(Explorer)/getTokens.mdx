## Get Tokens

<sub><i>async</i></sub> `getTokens(oktoClient: OktoClient)` retrieves the list of supported tokens.

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `oktoClient` | `OktoClient` | Instance of OktoClient obtained from useOkto hook |

### Response

<Callout title="Success Response">

| Field Name | Type | Description |
|------------|------|-------------|
| `tokens` | [`Promise<Token[]>`](/docs/react-sdk/technical-reference#models) | Array of supported tokens |

</Callout>

```typescript
import { useOkto } from '@okto_web3/react-sdk';
import { getTokens } from '@okto_web3/react-sdk';

function TokensList() {
    const oktoClient = useOkto();

    async function fetchTokens() {
        try {
            const tokens = await getTokens(oktoClient);
            console.log('Supported tokens:', tokens);
        } catch (error) {
            console.error('Error fetching tokens:', error);
        }
    }

    return (
        <button onClick={fetchTokens}>
            Fetch Supported Tokens
        </button>
    );
}
```
