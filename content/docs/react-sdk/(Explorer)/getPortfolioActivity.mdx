## Get Portfolio Activity

<sub><i>async</i></sub> `getPortfolioActivity(oktoClient: OktoClient)` retrieves the user's portfolio activity history.

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `oktoClient` | `OktoClient` | Instance of OktoClient obtained from useOkto hook |

### Response

<Callout title="Success Response">

| Field Name | Type | Description |
|------------|------|-------------|
| `activities` | `Promise<UserPortfolioActivity[]>` | Array of user's portfolio activities |

</Callout>

### Example

<Accordions>
<Accordion title="Usage">
<Tabs items={['TypeScript']}>
<Tab value="TypeScript">
```typescript
import { useOkto } from '@okto_web3/react-sdk';
import { getPortfolioActivity } from '@okto_web3/react-sdk';

function PortfolioActivity() {
    const oktoClient = useOkto();

    async function fetchActivity() {
        try {
            const activities = await getPortfolioActivity(oktoClient);
            console.log('Portfolio activities:', activities);
        } catch (error) {
            console.error('Error fetching activities:', error);
        }
    }

    return (
        <button onClick={fetchActivity}>
            Fetch Portfolio Activity
        </button>
    );
}
```
