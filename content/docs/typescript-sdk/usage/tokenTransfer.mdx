## Token Transfer

<sub><i>async</i></sub> `tokenTransfer(OktoClient, data)` transfers a token.

### Parameters

| Parameter   | Type                        | Description                            | 
|-------------|-----------------------------|----------------------------------------|
| `oktoClient` | [`OktoClient`](/docs/typescript-sdk/technical-reference#models) | Okto client |
| `data` | [`TokenTransferIntentParams`](/docs/typescript-sdk/technical-reference#models) | The data for the token transfer |

### Response

<Callout title="Success Response">

| Field Name | Type                  | Description                                   | 
|------------|-----------------------|-----------------------------------------------|
| `result`   | [`Promise<UserOp>`](/docs/typescript-sdk/technical-reference#models) | Returns a UserOp object for token transfer |
