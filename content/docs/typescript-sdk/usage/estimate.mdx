## Estimate

<sub><i>async</i></sub> `estimate(OktoClient, recipientWalletAddress, networkId, tokenAddress, amount)`<a href="https://github.com/okto-hq/okto-sdk-react/blob/main/src/OktoProvider.tsx#L340" target="_blank" rel="noopener noreferrer" style= {{ textDecoration: "none" }}> <IIcon icon="lucide:external-link" height="17" style={{ display: 'inline-block', verticalAlign: 'middle' }} /></a> estimates a user operation.

### Parameters

| Parameter   | Type                        | Description                            | 
|-------------|-----------------------------|----------------------------------------|
| `oktoClient` | `OktoClient` | Okto client |
| `recipientWalletAddress` | `string` | The recipient wallet address |
| `networkId` | `number` | The network id of the user operation |
| `tokenAddress` | `string` | The token address of the user operation |
| `amount` | `string` | The amount of the user operation |

### Response

<Callout title="Success Response">

| Field Name | Type                  | Description                                   | 
|------------|-----------------------|-----------------------------------------------|
| `result`   | [`Promise<EstimateOrderPayload>`](/api-docs#model/estimateorderpayload) | Returns the estimate order payload |
