## Connecting External Wallets with Okto

In addition to embedded wallets, Okto supports connecting user-owned wallets. This includes:

- **Browser Extensions:** MetaMask, Phantom, etc.
- **Hardware Wallets:** Ledger
- **Mobile Wallets:** WalletConnect-compatible apps
