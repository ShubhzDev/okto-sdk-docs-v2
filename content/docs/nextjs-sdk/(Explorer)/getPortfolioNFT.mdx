## Get Portfolio NFT

<sub><i>async</i></sub> `getPortfolioNFT(oktoClient: OktoClient)` retrieves the user's NFT balances.

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `oktoClient` | `OktoClient` | Instance of OktoClient obtained from useOkto hook |

### Response

<Callout title="Success Response">

| Field Name | Type | Description |
|------------|------|-------------|
| `nfts` | `Promise<UserNFTBalance[]>` | Array of user's NFT balances |

</Callout>

### Example

<Accordions>
<Accordion title="Usage">
<Tabs items={['TypeScript']}>
<Tab value="TypeScript">
```typescript
import { useOkto } from '@okto_web3/react-sdk';
import { getPortfolioNFT } from '@okto_web3/react-sdk';

function NFTPortfolio() {
    const oktoClient = useOkto();

    async function fetchNFTs() {
        try {
            const nfts = await getPortfolioNFT(oktoClient);
            console.log('NFT portfolio:', nfts);
        } catch (error:any) {
            console.error('Error fetching NFTs:', error);
        }
    }

    return (
        <button onClick={fetchNFTs}>
            Fetch NFT Portfolio
        </button>
    );
}
```
